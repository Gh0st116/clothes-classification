# clothes-classification
Deep learning model that classifies clothes into 10 categories during a course from Alura.

This model was made with the intention of helping a fictional e-commerce shop to automatize the categorization of its clothing.
<br><br>
Technologies used:
  - Tensorflow
  - Keras
  - Matplotlib
  - NumPy

Dataset used:
https://github.com/zalandoresearch/fashion-mnist
